【势力行动时钟：全书】

用途：
本条目执行任务清单阶段F。它让各势力在玩家不介入时继续行动，避免世界只围着玩家静止。行动时钟可被玩家提前触发、拖延、破坏或改写。

总部行动时钟：
1. 接线与记录：接线员收集地点、时间、人员、异常表现，先形成低权限记录。
2. 初步备案：若出现死亡、失踪或无法解释现象，生成事件编号和临时等级。
3. 地方封锁：警方、负责人或临时小队隔离现场，普通人口径被统一。
4. 资源审批：鬼烛、黄金容器、卫星定位手机、替死娃娃等按权限调拨。
5. 负责人任命：事件多发城市会设立负责人，玩家可被观察、招揽或限制。
6. 队长介入：当事件超过负责人处理能力，进入队长/总部会议层。
7. 最高封存：涉及民国旧人、终局、番外、阴阳路和旧名新名时，转入最高权限。

朋友圈/民间驭鬼者行动时钟：
1. 旁观与报价：先判断事件是否有利润、厉鬼拼图或灵异物品。
2. 招揽与试探：对新驭鬼者、负责人和玩家释放交易条件。
3. 情报收买：通过内线、助理、地方势力获取总部/玩家信息。
4. 围杀或清算：利益冲突升级后用灵异物品、暗杀、舆论和人情债施压。
5. 退场与重组：核心被清理后，残余成员改名、投靠或转入地下。

灵异论坛行动时钟：
1. 传闻收集：关注新事件、新驭鬼者和高价值厉鬼。
2. 交易撮合：提供情报、物品、保护或民间人脉。
3. 叶真关注：玩家挑衅、大海市事件或顶级战力传闻会吸引叶真。
4. 公开震慑：论坛以叶真名义压制民间圈，不等于听命总部。
5. 独立态度：国际冲突或总部行动中，论坛按自身利益决定介入程度。

鬼邮局行动时钟：
1. 选中新信使：玩家接触信件、邮局或相关地点后可能被纳入任务。
2. 送信期限：时间压力推动玩家进入危险地点。
3. 楼层晋升：完成/失败任务会改变信使层级和接触信息。
4. 管理者痕迹：高层任务逐渐暴露邮局管理者和油画世界。
5. 终局关闭：玩家或正史人物改变管理者结构后，信使时代进入收束。

太平古镇/招魂一脉行动时钟：
1. 地方规矩观察：外来者先被规矩和居民态度筛查。
2. 遗物媒介显露：出现招魂、鬼湖、鬼街、民国旧物线索。
3. 传承者介入：何银儿或相关人物评估是否动用招魂。
4. 高危亡魂回声：队长级或终局场景才可触发民国旧人回声。
5. 传承断绝/后日谈：番外线只在未来或高权限档案中开启。

国王组织行动时钟：
1. 海外情报收集：观察总部队长、鬼湖、大洪水计划和国内资源。
2. 试探与猎杀：派出国王或外围势力针对队长与关键人物。
3. 方舟/幽灵船推进：用幽灵船作为全球级压力筹码。
4. 和谈与拖延：在局势不利时以谈判争取登陆、调度或偷袭窗口。
5. 全面冲突：队长集结、鬼湖、招魂、公交和船长规则进入正面碰撞。
6. 残余处理：国王死亡或组织崩溃后，仍可能留下物品、档案和复仇者。

幽灵船行动时钟：
1. 船影传闻：海域异常、失踪船只、国外档案出现。
2. 航线接近：航海图、船舵、船舱和船长线索逐步显露。
3. 登陆压力：沿海封锁、鬼湖反制、国王组织协同。
4. 船上释放：大量厉鬼和船舱规则外溢。
5. 船长处理：进入最高难度，不允许普通手段解决。
6. 残留余波：战后仍有船上物品、海域污染和失踪者档案。

行动时钟推进条件：
- 玩家长时间不处理事件：时钟推进。
- 玩家公开情报：相关势力提前介入。
- 玩家使用高价值灵异物品：民间圈、总部或国王组织可能关注。
- 玩家提前改变正史：对应势力时钟跳步或改线。

使用规则：
- 每轮生成剧情时，若玩家没有主动处理某势力压力，允许后台推进一小格。
- 推进结果必须体现在可见线索、任务通知、敌对行动或封锁变化中。
- 不得让势力凭空知道玩家隐秘行动；需要情报来源、监控、内线或灵异媒介。
