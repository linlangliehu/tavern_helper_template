【鬼奴与衍生物规则｜v1】

鬼奴与衍生物是源头厉鬼、鬼域、媒介、杀人规律、污染或拼图成长产生的附属灵异个体。它们不是普通活人、普通尸体、普通精神控制者，也不等同于源头厉鬼本体。处理鬼奴只能减缓扩散、争取时间或获得线索，通常不能终结灵异事件。

一、基本定义
1. 鬼奴：由厉鬼杀死、污染、标记或在鬼域内转化出的附属个体。鬼奴通常没有正常理智，行为受源头鬼、鬼域、媒介或固定规律驱动，可能保留生前外貌、声音、动作习惯或身份假象。
2. 衍生物：由厉鬼规律产生的子鬼、影子、分身、复制体、污染人、档案人、空位人、媒介残留、鬼婴/鬼童类成长个体等。衍生物可能比普通鬼奴更接近独立厉鬼，但仍要先判断是否有源头、媒介或拼图关系。
3. 源头鬼：鬼奴或衍生物背后的核心灵异。源头可能是厉鬼本体、鬼域核心、媒介物、房间、古宅、梦境、画、档案、尸体、影子或某个拼图组合。
4. 鬼奴不等于弱小。单个鬼奴通常低于源头恐怖程度，但数量、鬼域、媒介传播、伪装、复活/刷新和群体行动会显著提高事件风险。

二、生成来源
1. 鬼域生成：能展开鬼域或规则空间的厉鬼，可能把死者、迷失者或被标记者转化为鬼奴。鬼域越稳定、越封闭、越深，鬼奴密度和刷新概率越高。
2. 杀人规律生成：被规律杀死的人可能成为鬼奴、媒介残留或下一轮袭击节点。例如听到、看见、回应、签字、照镜、入梦、开门、被影子覆盖等条件都可成为转化入口。
3. 污染/感染生成：饥饿、尸水、梦境、记忆、影子、血迹、照片、档案、编号、缺口、门牌等媒介可逐步污染活人，使其变成半鬼奴、标记者或衍生体。
4. 拼图成长生成：成长型厉鬼可能吞噬拼图、吸收尸体、复制标记者、孕育子鬼或制造附属个体。此类衍生物要记录成长阶段，不要一次性明牌完整机制。
5. 规则刷新生成：部分鬼奴不是一次性产生，而是会在固定时间、固定房间、梦境第二天、鬼域重启、尸体恢复或媒介重新出现时刷新。

三、识别原则
1. 可见异常优先：首次遇到只能写玩家看见的异常，例如动作重复、脸色死灰、没有呼吸、声音失真、机械服从、无痛觉、记忆错乱、影子多出、尸体行动、同一人反复出现。
2. 不直接明牌：没有证据时不得直接说“这是鬼奴”或“源头在某处”。可写“疑似被灵异控制”“像是尸体在行动”“可能只是外围异常”。
3. 区分活人和鬼奴：鬼奴可能保留生前外貌和求救话术，但不能把外貌、亲属关系、熟人声音当作安全依据。必须看是否有体温、呼吸、影子、记忆连续性、伤口反应、规律重复和媒介痕迹。
4. 区分鬼奴和源头：鬼奴会暴露规律痕迹，但通常不是最终本体。若清理后异常继续刷新、数量增加、鬼域不退或媒介仍活跃，应判定源头未被处理。
5. 区分衍生物和独立厉鬼：衍生物若开始拥有稳定杀人规律、独立鬼域、拼图吸附或脱离源头行动能力，可升级为“疑似独立厉鬼/高危衍生体”，但仍需证据确认。

四、行动与对抗
1. 普通武器通常只能破坏肉体表现，不能真正消灭鬼奴背后的灵异。对鬼奴开枪、焚烧、撞击或肢解，最多争取时间，可能触发媒介扩散、尸体刷新或源头锁定。
2. 黄金可隔绝部分媒介、尸体、标记物或小型衍生物，但不能自动关押源头。若鬼奴由鬼域或规则刷新产生，单纯封存一个个体无法结束事件。
3. 驭鬼者能力可压制、驱散、隔离或短暂清理鬼奴群，但必须结算复苏风险。若鬼奴数量很大，清理外围会快速消耗状态，不能替代寻找源头。
4. 灵异物品可制造窗口，例如鬼烛拖延、替死娃娃挡一次袭击、档案/照片/镜子类物品追踪媒介，但必须记录消耗和副作用。
5. 处理优先级：先保护活人并隔离媒介，再验证鬼奴行为规律，然后寻找源头、刷新点、控制媒介或拼图关系。不要把“杀掉所有鬼奴”写成默认胜利条件。

五、风险与灾害判定
1. 鬼奴数量少且范围小：可作为低到中风险外围异常处理，但接触、误认熟人、靠近尸体和追逐鬼奴仍可能触发规律。
2. 鬼奴数量增长或持续刷新：必须提高事件扩散趋势、死亡风险和总部关注度，优先判断是否存在源头鬼域、污染媒介或成长型厉鬼。
3. 鬼奴具备组织性、伪装性或跨区域传播：灾害等级可高于单体恐怖程度，因为风险来自传播、封锁压力、社会失控和误判。
4. 鬼奴能携带杀人规律：即使单个鬼奴不强，只要它能传播媒介、诱导回应、带路进鬼域、重复源头动作或成为刷新锚点，就必须按高危线索处理。
5. 清理鬼奴后的短暂安全不等于事件结束。若源头未关押，必须保留后续刷新、污染转移、隐藏标记者和灾害反弹的可能。

六、与鬼域、重启和拼图联动
1. 鬼域内鬼奴密度上升，通常代表玩家正在深入核心区、出口被改写、源头接近或规则正在刷新。按“鬼域规则｜v6”同步提高鬼域风险。
2. 鬼域重启、场所重启、梦境重启或规律循环可能让已清理鬼奴重新出现。重启来源和范围按“鬼域规则｜v6”记录。
3. 多鬼复合鬼域中的鬼奴可能继承多只厉鬼的媒介和规律，例如影子鬼奴带档案编号、手印残影携带被窃规律、尸体鬼奴携带梦境污染。必须判断同源、互补、冲突或失控。
4. 拼图吸附会改变鬼奴性质。源头厉鬼获得拼图后，鬼奴可能从普通尸体行动升级为子鬼、编号人、影奴、空位人、鬼婴或更高危衍生体。

七、开局预设厉鬼联动
1. 鬼手印：可衍生“冰冷手印/灵异空壳/残片鬼奴”。被鬼手印接触过的厉鬼、鬼奴、灵异媒介或驭鬼者体内灵异，可能被窃取一部分灵异力量并留下冰冷手印；被留下手印的目标可能短暂出现规律残缺、鬼域破损、媒介失效、复苏失衡或灵异反应迟滞。鬼手印获得的不是完整厉鬼本体，而是残缺灵异拼图；拼图方向偏向被窃取的规律片段、鬼域碎片、媒介能力、压制方式和复苏代价。
2. 鬼档案：可衍生“档案奴/编号人”。被归档者可能按编号行动、重复档案记载内容，或成为新的索引媒介。鬼档案的核心拼图不是普通纸张、签字或印章，而是“收录的厉鬼档案”。当鬼档案完整记录并验证一只厉鬼的杀人规律、媒介、触发条件、规避/压制窗口、鬼域表现、拼图关系和关押条件后，可生成一份厉鬼档案拼图；档案越完整，鬼档案恐怖程度越高。
3. 鬼拓本：可衍生“拓印奴/复写页/墨迹残身”。接触厉鬼规律、受害痕迹、灵异媒介或鬼域残留后，可能在纸面、皮肤、墙面或影像中留下拓印纹路，并复刻目标的一部分杀人规律。鬼拓本获得的不是源头鬼本体，而是“规律拓印”；同类型规律可互相覆盖、融合并进阶，不同类型规律可被撕碎、剥离或分解成灵异墨迹，用来提高鬼拓本恐怖程度。

七点一、鬼手印残片拼图机制
1. 窃取条件：鬼手印必须通过直接接触、被接触、媒介互触、鬼域交汇或灵异对抗窗口，才能窃取目标的一部分灵异力量。隔空知道名字、听说规律或看到现象不能完成窃取。
2. 窃取对象：可窃取厉鬼、鬼奴、衍生体、灵异媒介、灵异物品或驭鬼者体内厉鬼的残片力量；但默认只能取得残缺拼图，不等于夺走源头鬼本体。
3. 拼图成长：窃取到的灵异残片可作为鬼手印拼图，提升鬼手印恐怖程度，并让鬼手印短暂或有限调用目标的规律片段、媒介能力、压制方式或鬼域碎片。残片越完整、来源越强，成长越明显，失控风险也越高。
4. 被窃后果：被窃取目标可能短暂规律残缺、媒介失灵、鬼域破损、复苏失衡或反向锁定鬼手印；若目标恐怖程度远高于鬼手印，窃取可能失败并触发反噬。
5. 鬼域叠加：若鬼手印窃取到鬼域碎片，可形成“印域叠加”。这种叠加按“鬼域规则｜v6”判断同源、互补、冲突或单向压制；残缺鬼域通常不稳定，可能只表现为空间破洞、短暂通道、错位边界或局部压制。
6. 平衡与死机：鬼手印可通过多份残片互相牵制形成临时平衡；若玩家拥有死机驾驭或已让鬼手印死机，调用残片时可降低自身复苏代价。但残片本身仍可能复苏、污染宿主或互相冲突。
7. 禁止无上限复制：鬼手印不能无条件复制完整厉鬼能力，不能凭一次接触永久获得完整规律，也不能无视恐怖程度差距窃取世界级厉鬼。每次窃取必须记录接触证据、目标来源、残片完整度、可调用范围和反噬风险。

七点二、鬼档案拼图机制
1. 收录条件：鬼档案不能凭空收录厉鬼。必须通过观察、试探、死亡案例、总部档案、玩家可见证据或灵异代价，逐步验证该厉鬼的完整杀人规律、触发媒介、规避方式、压制窗口、鬼域/异常空间表现、拼图关系和关押条件。只知道名字、传闻或表面现象时，只能生成残缺档案，不能作为稳定拼图。
2. 档案拼图：完整厉鬼档案本身就是鬼档案的拼图。每收录一份稳定档案，鬼档案的恐怖程度、规则完整度和可调用能力都会上升。残缺档案只能提供线索、弱化投影或错误诱导；完整档案可调用明确规律；源头档案或关押档案稳定度最高。
3. 规律调用：鬼档案可以使用档案厉鬼的部分杀人规律或媒介规则，但必须受档案完整度限制。未验证的规律不得稳定调用；被误判写入的规律可能反向误导玩家、制造错误档案或触发档案污染。
4. 鬼域叠加：若档案中已验证对应厉鬼具备鬼域、规则空间、媒介空间或空间干涉能力，鬼档案可以调用该档案鬼域，与鬼档案自身档案鬼域形成复合鬼域。复合方式按“鬼域规则｜v6”判断同源、互补、冲突或单向压制，不能简单相加层数。
5. 代价转嫁：鬼档案使用灵异时，代价优先由被调用的档案厉鬼承担，表现为档案渗血、字迹涂改、编号错乱、档案厉鬼复苏痕迹增强或对应媒介异常。鬼档案本体可因此保持近似死机，但不是无条件免费；过度调用会让档案污染、档案厉鬼反向苏醒或鬼档案鬼域失控。
6. 平衡与死机：多份档案厉鬼之间可互相牵制，形成动态平衡；鬼档案本体负责收录、编号、调度和调用，受到代价转嫁保护时可维持近似死机。若某份档案恐怖程度远高于其它档案，或档案规律互补过强、冲突过烈，平衡会被打破。
7. 档案不是本体：被归档的厉鬼档案不等于源头鬼本体被永久奴役。源头鬼仍可能在外界行动、被关押、被转移或通过媒介反向污染档案。清理档案投影不能替代关押源头。
8. 高危收录：试图收录鬼差、饿死鬼、鬼画、鬼湖、鬼梦、鬼邮局等高恐怖程度或世界级厉鬼时，必须提高风险。档案可能成为源头入口，导致鬼档案被压制、档案库被污染、玩家被编号锁定或复合鬼域失控。
9. 灵异档案视野联动：若玩家选择“灵异档案视野”特殊能力，它优先服务于鬼档案收录流程。该视野可以提示档案完整度、已验证字段、缺失字段、疑似媒介、疑似触发条件、鬼域迹象、可收录风险和档案拼图稳定度；但不能直接公开完整隐藏规律、源头位置、最终生路或无条件关押方案。提示必须以玩家已接触证据为基础，未验证内容只能标注为“疑似/缺失/待验证”。

七点三、鬼拓本规律拓印机制
1. 拓印条件：鬼拓本必须接触、承受或近距离覆盖目标厉鬼的规律表现，或接触明确的受害痕迹、灵异媒介、鬼域残留、关押物污染、鬼奴残留，才能形成规律拓印。只知道名字、听说档案或远距离猜测，不能生成稳定拓印。
2. 复刻范围：鬼拓本默认只能复刻目标杀人规律、媒介规则、触发方式、压制方式或鬼域表现的一部分，不等于夺走源头鬼本体，也不等于完全掌握目标全部规律。拓印越完整，调用越稳定，反噬也越强。
3. 同类融合：同类型规律可以融合进阶。例如同属“看见触发”“声音触发”“接触标记”“媒介锁定”“鬼域封锁”的拓印，可叠加成更强、更完整、更难规避的规律。融合后鬼拓本恐怖程度、规则完整度和压制力上升，但复苏污染也会集中。
4. 异类分解：不同类型且无法兼容的规律不能简单混用，可被分解成灵异墨迹、残页、拓纹或污染笔画，用于提高鬼拓本恐怖程度、补全规则空白或强化拓本媒介。分解会损失原规律细节，不能同时保留完整调用权。
5. 拓本拼页：每份规律拓印都应记录来源、类型、触发条件、媒介、完整度、融合/分解状态和反噬风险。拓印页越多，鬼拓本越强，也越容易出现错页、重影、错误触发和规律互相覆盖。
6. 鬼域联动：若拓印目标具备鬼域、规则空间或媒介空间，鬼拓本可以获得残缺鬼域拓印。残缺鬼域可与其它同类空间规律融合，但按“鬼域规则｜v6”判定同源、互补、冲突或单向压制，不能直接复制完整高层鬼域。
7. 平衡与死机：鬼拓本可用同类融合形成稳定主规律，也可用异类分解降低冲突。若玩家拥有“永久死机驾驭”，鬼拓本本体会死机，调用已驾驭本体能力无自身复苏代价；但拓印规律来源的污染、错页、反向覆盖和外来灵异反噬仍需单独判定。
8. 禁止无上限复刻：鬼拓本不能无条件复制世界级厉鬼，不能凭一次拓印永久获得完整规律，不能把互相冲突的规律全部无代价融合。高恐怖程度目标可导致拓印失败、拓本被污染、规律反向写入玩家或源头鬼通过拓本追踪玩家。
9. 规律目录视野联动：若玩家选择“规律目录视野”特殊能力，它优先服务于鬼拓本的拓印整理流程。该视野可以提示规律类型、拓印完整度、兼容方向、可融合/可分解倾向、疑似媒介、污染风险、反噬风险和推荐处理；但不能直接公开完整隐藏规律、源头位置、最终生路、关押方案或世界级厉鬼完整拼图。提示必须以玩家已接触证据为基础，未验证内容只能标注为“疑似/缺失/待验证”。

八、状态与数据库联动
1. 当前灵异事件可记录“鬼奴状态：未确认/疑似/少量/增长/密集/持续刷新/失控扩散”。玩家未验证前，正文和状态栏只写可见迹象。
2. 规律推理记录应记录鬼奴证据，例如“尸体仍在行动”“同一个人第二次出现”“被击倒后在原房间刷新”“影子脱离后独立行动”“编号者按档案顺序敲门”。
3. 隐藏档案可记录源头控制方式、鬼奴刷新点、真实媒介、转化条件、是否可逆、是否已独立、是否存在拼图升级，但不得直接泄露给玩家。
4. 鬼奴或衍生物被清理、封存、失控或升级时，应同步更新当前灵异事件、厉鬼档案、规律推理记录、死亡人数、扩散趋势、地点状态和世界线记录。

九、叙事边界
1. 鬼奴用于制造压迫、误判、扩散和线索，不应每次都写成正面战斗杂兵。
2. 熟人变成鬼奴时，重点写玩家可见违和感和选择压力，不要用旁白直接解释完整机制。
3. 清理外围鬼奴可以给玩家阶段性收益，例如救出幸存者、拿到媒介、验证规律或争取撤离窗口，但必须留下源头未解的压力。
4. 若玩家拥有“永久死机驾驭”等外挂，玩家已经成功驾驭的所有厉鬼本体都会死机，可免除这些已驾驭厉鬼本体能力带来的自身用鬼代价；但后续新厉鬼仍必须先完成驾驭，鬼奴、残片、档案、媒介或交易引入的外来灵异仍需单独结算污染、冲突、反噬和失控风险，不能自动辨认所有鬼奴、直接找到源头或阻止鬼奴刷新。
