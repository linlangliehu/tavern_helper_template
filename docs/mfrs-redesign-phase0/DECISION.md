# Phase 0 · 路径决策（T0.3）

## 已采纳：路径 α（消息内仿三栏）— 第一期

```
┌──────────────┬────────────────────────────┬────────┐
│ 现场档案     │ 坐标·阶段·事件·危害        │ 导航   │
│ (可折叠)     │────────────────────────────│ 正文   │
│ 身份/风险/   │     AI 叙事正文            │ 档案   │
│ 事件/厉鬼…   │     拟办意见 A/B/C/D       │ 关系   │
│              │                            │ 柜     │
└──────────────┴────────────────────────────┴────────┘
```

| 决策 | 选择 | 理由 |
|------|------|------|
| 布局落地 | **路径 α**：在最新 AI 消息内 CSS Grid 三栏 | 不改 ST 全局布局；可随角色卡发版；风险可控 |
| 固定 host | **Phase3 仅换皮** | 保留 `insertBefore` + order 10/20 |
| Brand | **删除**眼/阵 +「鬼眼封案」主视觉 | 对齐参考图无神器 Logo |
| 顶条 | 位置·阶段·事件代号·鬼域·危害 | 只读现有 `stat_data` |
| 行动 | 样式改「拟办意见」；**仍只填不发** | 契约不变 |
| 路径 β | **延期** | 左/右栏常驻 host 为二期可选 |

## 用户确认状态
