# 神秘复苏模拟器 8.13.8

**日期**：2026-07-12  
**主题**：无真实行动建议时隐藏本轮选项栏（去掉开局占位 A–D）

## 问题

开局第 0 楼 `行动建议=[]` 时，HUD 仍展示硬编码占位（走廊敲门声等），易被误认为已开局/串聊。

## 修复

- 仅从 MVU `行动建议` 或数据库「行动建议」表收集真实项
- 无真实数据：`[data-mfrs-hud="actions"]` 整栏 `hidden`，槽清空
- 有落库数据：再显示 A–D（点选只填输入框）
- 删除 provisional 开局占位文案
