# 神秘复苏模拟器 8.13.6

**日期**：2026-07-12  
**主题**：热修 8.13.5 外置面板扫描误伤 ST 核心布局导致沉浸闪烁

## 问题

8.13.5 兜底扫描把 `#bg1` / `#sheld` 当成「外置大面板」抬到 z=10080，随后因 z≥10080 又被排除 → `yield`/`restore` 死循环，表现为原生布局与沉浸画面快速切换。

## 修复

- 明确排除 ST 核心布局：`#bg1` `#sheld` `#chat` `#form_sheld` 等
- 不再对 body 子级做无白名单面积扫描误伤
- `bindHudShellEvents` 挂载时不主动 yield
- MutationObserver 忽略核心布局抖动；仅扩展入口 / 真实外置面板时让层
