# Phase 34-H 实机对比验证清单

## 验证目标
对比神秘复苏模拟器和蛊真人 v10.2 的仪表盘，确认神秘复苏已完全对齐 v10.2 的结构和布局（保留暗红主题）。

## 验证步骤

### 1. 加载角色卡
- [ ] 在 SillyTavern 中加载 `dist/神秘复苏模拟器.png`
- [ ] 在另一个浏览器标签页或窗口中加载 `v10.2.png`（蛊真人 v10.2）
- [ ] 确认两个角色卡都已加载星河璀璨·数据库扩展

### 2. 触发仪表盘显示
- [ ] 神秘复苏：发送一条消息，等待 AI 回复，确认仪表盘出现
- [ ] 蛊真人 v10.2：发送一条消息，等待 AI 回复，确认仪表盘出现

### 3. 挂载位置对比
- [ ] 神秘复苏仪表盘是否挂在最后一条 AI 消息块内？
- [ ] 蛊真人 v10.2 仪表盘是否挂在最后一条 AI 消息块内？
- [ ] 两者挂载位置是否一致？

### 4. 容器结构对比
- [ ] 神秘复苏：打开浏览器开发者工具，检查仪表盘容器类名是否为 `.acu-embedded-dashboard-container`
- [ ] 蛊真人 v10.2：检查仪表盘容器类名是否为 `.acu-embedded-dashboard-container`
- [ ] 两者容器结构是否一致？

### 5. 布局对比
- [ ] 神秘复苏：仪表盘是否为三列布局？
- [ ] 蛊真人 v10.2：仪表盘是否为三列布局？
- [ ] 神秘复苏：第一列是否显示"全局状态"和"玩家状态"？
- [ ] 蛊真人 v10.2：第一列是否显示类似的状态卡片？
- [ ] 神秘复苏：第二列是否有"灵异事件"和"线索" tab 切换？
- [ ] 蛊真人 v10.2：第二列是否有类似的 tab 切换？
- [ ] 神秘复苏：第三列是否有"厉鬼档案"和"数据库" tab 切换？
- [ ] 蛊真人 v10.2：第三列是否有类似的 tab 切换？

### 6. 主题颜色对比
- [ ] 神秘复苏：仪表盘是否为暗红恐怖风格（暗红背景、血红高亮、灰白文本）？
- [ ] 蛊真人 v10.2：仪表盘是否为 aurora 风格（蓝紫色）？
- [ ] 确认：神秘复苏保留了暗红主题，没有复制 v10.2 的 aurora 颜色 ✓

### 7. MVU 状态整合验证
- [ ] 神秘复苏："玩家状态"卡片是否显示"死亡风险""复苏风险""最近行动""行动建议"等字段？
- [ ] 这些字段的值是否来自 MVU `stat_data` 即时状态（而不是数据库表）？
- [ ] 如果数据库表为空，是否仍然显示 MVU 即时状态？

### 8. 交互功能验证
- [ ] 神秘复苏：点击仪表盘顶部控制栏，是否可以折叠/展开？
- [ ] 神秘复苏：点击 tab 按钮，是否可以切换内容？
- [ ] 神秘复苏：点击"刷新"按钮，是否可以刷新数据？
- [ ] 神秘复苏：点击"控制台"按钮，是否可以打开大控制台？
- [ ] 神秘复苏：点击"编辑"按钮，是否可以打开数据库编辑器？

### 9. 移动端布局验证（可选）
- [ ] 将浏览器窗口缩小到 500px 宽度
- [ ] 神秘复苏：仪表盘是否自适应为单列布局？
- [ ] 蛊真人 v10.2：仪表盘是否自适应为单列布局？
- [ ] 两者移动端布局是否一致？

### 10. 截图对比（可选）
- [ ] 截图神秘复苏仪表盘（桌面端）
- [ ] 截图蛊真人 v10.2 仪表盘（桌面端）
- [ ] 对比两张截图，记录差异点

- ✅ 布局一致（三列、卡片、tab 切换）
